# March Madness Machine Learning Project
A collection of models trained on past NCAA games to predict future winners.

## Technologies Used
- streamlit
- scikit-learn
- pandas
- numpy

## Models Used
- Support Vector Machine (Classifier)
- Random Forest Classifier
- Logistic Regression

## Features
- Dropboxes used to select teams
- Predict button uses majority vote of models
